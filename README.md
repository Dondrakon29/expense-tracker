# Expense Tracker

A simple Python console project for tracking personal expenses.

## Features

- Add new expenses
- Show all expenses
- Show expenses by category
- Delete expenses by number
- Save expenses to JSON file
- Load expenses from JSON file
- Validate user input with `try / except`
- Validate date format using `datetime`
- Calculate total expenses
- Basic code refactoring with helper functions
```markdown
- Clear all expenses with confirmation
## Technologies

- Python
- JSON
- Git
- GitHub

## How to run

1. Open the project folder.

2. Run the file:

```bash
python expense_tracker.py
```

## Menu

The application has a simple console menu:

```text
1 - Add expense
2 - Show expenses
3 - Show by category
4 - Delete expense
5 - Clear all expenses
0 - Exit
## Expense example

Each expense is stored as a dictionary:

```json
{
    "title": "Coffee",
    "amount": 150,
    "category": "Food",
    "date": "2026-06-10"
}
```
